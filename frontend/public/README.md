# Public Assets

This directory contains static assets that are served directly by the web server.

## Logo Files

- `quipo-logo.svg` - QUIPO Universal logo in SVG format
  - Can be used as: `<img src="/quipo-logo.svg" alt="QUIPO Logo" />`
  - Or imported in components: `import logo from '/quipo-logo.svg'`

## Usage

Files in this directory are accessible at the root path:
- `/quipo-logo.svg` - Logo file
- `/favicon.ico` - Favicon (if added)
- `/robots.txt` - Robots file (if added)

## Adding Assets

Place any static assets (images, fonts, etc.) in this directory and reference them with absolute paths starting with `/`.
